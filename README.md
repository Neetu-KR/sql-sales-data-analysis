# SQL Sales Data Analysis Portfolio

This repository features an interactive SQL practice project mapping real-world operations on an online mock sales dataset. 

## 🛠️ Operations Demonstrated
* **Database & Table Setup:** Table creation (`CREATE TABLE`) and seeding raw data (`INSERT INTO`).
* **Data Retrieval & Merging:** Combining distinct sales records and territorial tracking via `INNER JOIN`.
* **Conditional Mapping:** Writing dynamic segmentations with `CASE WHEN` clauses.
* **Data Manipulation:** Modifying specific database items with `UPDATE` and `DELETE`.

## 📁 Repository Contents
* `sales_analysis_project.sql`: The master executable database script.
